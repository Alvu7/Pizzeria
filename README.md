# Circuito Logisim en Web

## Ejecutar todo

```bash
python3 main.py
```

Abrir en navegador:

- http://localhost:8000

## Archivos

- `index.html`: diagrama completo del circuito
- `styles.css`: estilos de visualización
- `main.py`: servidor local
- `CIRCUITOS_LOGISIM.md`: guía textual del circuito
