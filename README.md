# Circuito Logisim en Web

## Ejecutar

```bash
python3 main.py
```

Abrir en navegador:

- http://localhost:8000

## Archivos finales

- `CIRCUITOS_LOGISIM.md` (guía del circuito)
- `index.html` (diagrama visual)
- `styles.css` (estilos)
- `main.py` (servidor local)

## Si GitHub marca conflictos

En el editor de conflictos de GitHub, para estos 4 archivos:
- `CIRCUITOS_LOGISIM.md`
- `README.md`
- `index.html`
- `main.py`

elige la versión más reciente de esta rama y guarda.
