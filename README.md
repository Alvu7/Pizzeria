# Simulador de Nota de Voz (VHDL + Web)

## Activar simulador web (main)

```bash
python3 main.py
```

Luego abre en el navegador:

- http://localhost:8000

## Archivos clave

- `index.html`: interfaz web del simulador
- `main.py`: **main** para activar servidor local
- `voice_note_player.vhd` y módulos `*.vhd`: implementación HDL modular
